# Space-Courier
Это моя вторая игра.Вот описание:«Космический курьер» — аркада на выживание в открытом космосе. Управляйте кораблём, собирайте 5 топливных капсул и уклоняйтесь от астероидов. Используйте бонусы (щит, ускорение, замедление времени). Вас ждёт пояс астероидов и посадка на Землю. Голосовой помощник Алиса даёт подсказки.Победа будет за вами!
